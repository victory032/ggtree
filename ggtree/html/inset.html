inset {ggtree}R Documentation

inset

Description

add insets in a tree

Usage

inset(tree_view, insets, width = 0.1, height = 0.1, hjust = 0,
  vjust = 0, x = "node", reverse_x = FALSE, reverse_y = FALSE)

Arguments

tree_view

tree view

insets

a list of ggplot objects, named by node number

width

width of inset

height

height of inset

hjust

horizontal adjustment

vjust

vertical adjustment

x

x position, one of 'node' and 'branch'

reverse_x

whether x axis was reversed by scale_x_reverse

reverse_y

whether y axis was reversed by scale_y_reverse

Value

tree view with insets
