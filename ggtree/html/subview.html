subview

Description

add subview to mainview for ggplot2 objects

Usage

subview(mainview, subview, x, y, width = 0.1, height = 0.1)

Arguments

mainview

main view

subview

a ggplot or grob object

x

x position

y

y position

width

width of subview, [0,1]

height

height of subview, [0,1]

Value

ggplot object
